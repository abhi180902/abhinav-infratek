# Abhinav Infratek Website Admin

This part of the project supports the Abhinav Infratek website admin area.

The admin area is created so the company owner can manage website content without changing code.

## Purpose

The admin website helps keep the public website updated with fresh company information, services, project details, gallery content, and customer enquiries.

## Admin Content Areas

The admin area is planned to manage:

- Projects
- Services
- Gallery
- Team Members
- Contact Messages
- Career Applications
- Blogs
- Certificates
- Company Information

## Owner Benefits

The admin area allows the owner to:

- Add new projects
- Update service details
- Manage gallery images
- View customer messages
- Maintain company information
- Keep website content updated easily

## Public Website Connection

The public website can show updated content from the admin area, especially services and projects. This helps the website stay useful as the company grows and completes more work.
